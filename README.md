# PASSWORD STRENGTH CHECKER

# Project Description:

Built using Python, this project checks the strength of the user entered password based on security rules such as the length, use of uppercase or lowercase letters, numbers and special characters.

The aim is to help users understand how secure their passwords are and encourage better password practices.

# Project Features:
    - Determines the strength of the password
    - Gives feedback on whether it is Weak, Medium or Strong
    - Helps user create better and more secure passwords 

# Working Of The Project:
    - Checks whether the password meets the minimum length requirement 
    - Verifies if the input includes Uppercase and Lowercase Letters
    - Checks if special characters(&,$,!,@) are used
    - Confirms if it contains numbers

Based on the above listed factors, the overall strength of the password is calculated and the final result is displayed to the user. 

# How To Run:
    -Download the file
    -Open a new terminal/command prompt
    -Run:
        python password_strength.py
